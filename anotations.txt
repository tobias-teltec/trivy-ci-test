Trivy vs Anchore
Clair:
- Clair clients depreciados
  Difícil instalação
  Documentação fraca

Anchore:
- Mais personalização no que diz respeito a policies
  Possui Helm para instalação no k8s
  Possui Anchore Admission Controller for k8s
  Possui plugin do Jenkins

Trivy:
- Mais simples de instalar
  Mais rápido
  Não precisa de banco de dados (stateless)
  Banco de dados CVE atualizado mais rapidamente e com mais frequência
  Trata a base de dados para evitar duplicidade (base menor em torno de 17MB)
  Opção de base de dados --light para CI-CD
  Maior precisão de vulnerabilidades principalmente para imagens Alpine 
  Starboard tool para scanear containers dentro do cluster k8s
